# Prompt Engineering Mastery

> **Course 1** — From casual AI user to systematic prompt engineer.

**Target Audience:** Non-technical professionals, marketers, researchers, analysts, and project managers.  
**Duration:** 4 Weeks (2 lessons per week) + Capstone Project.  
**Objective:** Move past "simple chatting" into systematic, predictable, and programmatic control of Large Language Models (LLMs).

## Models Covered

This course provides concrete examples and behavioral comparisons for:
- **OpenAI** — GPT-4o, GPT-4o-mini, o3/o3-mini (reasoning models)
- **Anthropic** — Claude 3.5 Sonnet, Claude 3 Opus (extended thinking)
- **Google** — Gemini 2.5 Pro, Gemini 2.5 Flash (massive context, multimodal)
- **Meta** — Llama 3.1 (open-source, 8B/70B/405B)
- **Mistral** — Mistral Large, Codestral
